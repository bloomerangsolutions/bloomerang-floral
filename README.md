# Bloomerang Floral — Full Template Site

27-page florist website template with full SEO, GEO, AEO optimization.

## Deploy to Vercel

```bash
cd bloomerang-floral
vercel --prod
```

When prompted:
- Team: **Bloomerang Solutions**
- Project name: **bloomerang-floral**
- Framework: **Other**
- Output directory: **.** (current directory)

## Pages
- Homepage, About, Shop (6 category pages), Occasions (6), Services (5), Help (5), SEO Geo Pages (3)
- 27 pages total, 1,530 internal links, 34 images
- Schema.org JSON-LD on every page
- Full GEO meta tags for Sarasota, FL

## Stack
- Pure HTML/CSS/JS — no framework, no build step
- Shared `style.css`
- Images in `/img/`
- Lora + DM Sans + JetBrains Mono typography
